List of all set-by-user shortcuts.

Type A:	aliases
Type S: scripts (they may have some options; try: -h, --help or -? to learn)
Type R: need root privileges (sudo command included)

 google		S	Google

 powerpoint	A	Microsoft Office PowerPoint Online
 udemy		A	Udemy
 usos		A	USOS AGH

Git commands:

 COMMAND	TYPE	DESCRIPTION
 pushCode	S	add chosen files to git & commit & push to GitHub

System commands:
 
 COMMAND	TYPE	DESCRIPTION

 suspend	A	works like: off --suspend
